## 🧠 summarize-file

Summarize any text file using **Gemini** or **OpenAI** just from CLI.

```bash
npx summarize ./file.txt
```

## 🧰 Requirements

- **Node.js ≥ 18**
- **Internet connection**
- A valid **Gemini** or **OpenAI** API key stored in your environment variables

## 🙅‍♀️ No API key?

```bash
npx summarize --setup # will be implemented soon
```
